    Descripción General

    Esta ventana permite al usuario elegir la acción que desea realizar en el programa. Contiene cuatro opciones representadas por iconos, cada una con una función específica: FTP, correo electrónico, configuración y consulta de información de la empresa.

    Componentes de la Ventana

    1. Opción: FTP
      • Ubicación: Parte superior izquierda.
      • Descripcion: Representado por un icono con un reptil y una carpeta.
      • Función: Permite al usuario acceder al sistema FTP para gestionar archivos mediante transferencia de datos.
    2. Opción: Correo Electrónico
      • Ubicación: Parte superior derecha.
      • Descripcion: Representado por un icono de una arroba (@).
      • Función: Permite al usuario acceder al sistema de correos para enviar y recibir mensajes electrónicos.
    3. Opción: Configuración
      • Ubicación: Parte inferior izquierda.
      • Descripcion: Representado por un engranaje con una figura de usuario en el centro.
      • Función: Permite al usuario gestionar la configuración del programa, como ajustes de sistema y personalización.
    4. Opción: Información
      • Ubicación: Parte inferior derecha.
      • Descripcion: Representado por un icono de una letra i (información).
      • Función: Proporciona información sobre la empresa, como datos de contacto, detalles del software o manuales.

    Navegación

    • El usuario puede seleccionar cualquiera de las opciones haciendo clic en el icono correspondiente.
    • Una vez seleccionada una opción, el programa redirigirá a la funcionalidad elegida.
