    • Ventana del Sistema FTP

    Descripción General

    La ventana del sistema FTP permite al usuario realizar tareas de gestión de archivos y directorios. Estas acciones incluyen crear, eliminar, modificar y descargar archivos o carpetas. El árbol de archivos ubicado en la interfaz muestra la estructura actual del sistema, facilitando la navegación.

    Componentes de la ventana

    1. Barra Superior
      • Campo de texto: Ubicado en la parte superior central. Permite ingresar el nombre del archivo o carpeta que se desea renombrar.
      • Botón Rename:
        • Ubicación: Parte superior derecha.
        • Función: Actualiza y recarga el árbol de archivos para mostrar la estructura más reciente.
    2. Botón Reload
      • Ubicación: Parte superior izquierda.
      • Función: Actualiza y recarga el árbol de archivos para mostrar la estructura más reciente.
    3. Opciones de Gestión de Archivos/Directorios

      4. Ubicadas en la sección izquierda de la ventana, estas opciones permiten interactuar con el sistema FTP:

      • Create Directory:
        • Crea un nuevo directorio dentro de la ubicación seleccionada.
      • Delete Directory:
        • Elimina el directorio seleccionado en el árbol de archivos.
      • Modify Directory:
        • Permite modificar las propiedades o el nombre de un directorio existente.
      • Delete File:
        • Elimina un archivo seleccionado en el árbol de archivos.
      • Download File:
        • Descarga un archivo específico a una ubicación local.
    4. Árbol de Archivos (JTree)
      • Ubicación: Parte derecha de la ventana.
      • Descripción: Muestra la estructura del sistema de archivos FTP. Los directorios y subdirectorios se representan de forma jerárquica para facilitar la navegación.
    5. Botón Exit
      • Ubicación: Parte inferior central.
      • Función: Cierra la ventana FTP y finaliza la sesión actual.

    Usos de la ventana FTP

    1. Navegación:
      • Utilice el árbol de archivos para seleccionar el directorio o archivo con el que desea trabajar.
    2. Gestión de Archivos/Directorios:
      • Haga clic en las opciones del menú izquierdo para realizar las siguientes acciones:
        • Crear, eliminar o modificar directorios.
        • Eliminar o descargar archivos.
    3. Renombrar Archivos/Directorios:
      • Ingrese el nuevo nombre en el campo de texto superior.
      • Seleccione el elemento en el árbol de archivos.
      • Haga clic en Rename.
    4. Actualizar:
      • Haga clic en Reload para refrescar la vista y confirmar los cambios realizados.
    5. Salir:
      • Presione el botón Exit para cerrar la ventana.

    Propósito de la Ventana

    Facilitar al usuario la gestión remota de archivos y directorios en un sistema FTP mediante una interfaz gráfica intuitiva y funcional.
